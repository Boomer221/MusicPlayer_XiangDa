

##期末作业，做了个音乐播放器。

* 音乐绑定服务播放的，支持后台播放
* 搜索是全盘搜索，指定了MP3格式，并且筛选丢弃小于1MB的文件
* 歌词以及背景图需要自己下载，并且设定

有些简陋，仅供参考下。
